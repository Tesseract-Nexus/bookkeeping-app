# UI/UX Design System

## Design Philosophy

### Core Principles

1. **Simplicity First**
   - Every feature should be usable without reading instructions
   - Hide complexity until needed (progressive disclosure)
   - One primary action per screen

2. **Mobile-Native Thinking**
   - Design for thumb-first interaction
   - Optimize for one-handed use
   - Consider varied network conditions

3. **Trust Through Clarity**
   - Financial data displayed clearly with proper formatting
   - Consistent visual hierarchy
   - Immediate feedback for all actions

4. **Zero Learning Curve**
   - Use familiar patterns from popular apps
   - Natural language over accounting jargon
   - Contextual help when needed

## Interaction Patterns

### Gestures (Mobile)

| Gesture | Action |
|---------|--------|
| Tap | Select, navigate |
| Long press | Context menu, bulk select |
| Swipe left | Quick actions (delete, archive) |
| Swipe right | Quick actions (edit, mark paid) |
| Pull down | Refresh |
| Pull up | Load more |

### Feedback

```
Actions should provide immediate feedback:

1. Button Press
   - Visual: Scale down slightly (0.98)
   - Haptic: Light tap

2. Success
   - Toast: "Invoice created successfully"
   - Animation: Checkmark
   - Haptic: Success pattern
   - Duration: 2 seconds

3. Error
   - Toast: Error message (red)
   - Shake animation on field
   - Haptic: Error pattern
   - Persist until dismissed

4. Loading
   - Button: Show spinner, disable
   - Screen: Skeleton loading
   - Pull refresh: Native spinner
```

## Accessibility

### Requirements

1. **Color Contrast**: Minimum 4.5:1 for text, 3:1 for large text
2. **Touch Targets**: Minimum 44x44px
3. **Font Scaling**: Support up to 200% scaling
4. **Screen Readers**: All interactive elements labeled
5. **Keyboard Navigation**: Full keyboard accessibility on web

### Dark Mode

- Auto-detect system preference
- Manual toggle in settings
- All colors have dark mode variants
- Reduce eye strain for extended use

## Animation Guidelines

### Timing

```css
--duration-fast:   150ms;  /* Micro-interactions */
--duration-normal: 300ms;  /* Standard transitions */
--duration-slow:   500ms;  /* Complex animations */

--easing-default: cubic-bezier(0.4, 0, 0.2, 1);
--easing-enter:   cubic-bezier(0, 0, 0.2, 1);
--easing-exit:    cubic-bezier(0.4, 0, 1, 1);
```

### Motion Principles

1. **Purpose**: Every animation serves a purpose
2. **Natural**: Follow physics-based motion
3. **Quick**: Don't make users wait
4. **Subtle**: Enhance, don't distract

### Key Animations

- Page transitions: Slide/fade
- Modal: Slide up from bottom
- Toast: Slide in from top
- Success: Checkmark draw
- Delete: Fade out + slide
